# GSTGenie

🏆 **Winner – GOVERNMENT ENGINEERING COLLEGE, BHAVNAGAR OPEN INNOVATION CHALLENGE-2025**

GSTGenie is a web-based compliance automation platform developed as the winning solution for the **Startup Simplifier (PS06)** problem statement in the **Government Engineering College, Bhavnagar Open Innovation Challenge-2025**.

The project aims to simplify GST registration, startup compliance, and document management through a single, user-friendly platform. It provides guided GST registration, AI-powered assistance, an integrated document repository, and a chatbot to help entrepreneurs navigate compliance requirements with ease.

> **Note:** During development, the application was internally named **GSTease**. The project is now officially presented as **GSTGenie**.

## 📌 Problem Statement

**PS06: Startup Simplifier**

Many young entrepreneurs hesitate to start a business due to the complexity of GST registration, compliance, and legal documentation.

**Objective:** Develop a one-stop compliance automation platform that simplifies startup registration and GST-related processes.

## ✨ Implemented Features

* Digital GST registration assistant
* Step-by-step registration wizard
* AI-powered guidance throughout the registration process
* Integrated document repository
* Startup query chatbot
* User authentication
* Multi-language support
* Dark & Light theme
* Responsive user interface
* Registration progress tracking

## 🚧 Future Enhancements
